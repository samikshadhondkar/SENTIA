# 🌟 SENTIA

> **An AI-powered accessibility companion that helps visually impaired individuals navigate the world with greater confidence, independence, and safety.**

---

## 📖 Overview

SENTIA is an intelligent mobile assistant designed to empower people with visual impairments through real-time environmental awareness and voice-guided assistance. By combining artificial intelligence, computer vision, and intuitive user interactions, SENTIA transforms a smartphone into a reliable companion capable of understanding surroundings, identifying obstacles, and assisting users in everyday navigation.

Our goal is simple: **make independent mobility safer, smarter, and more accessible for everyone.**

---

## ✨ Features

* 🧠 AI-powered scene understanding
* 🚶 Intelligent obstacle detection
* 🗣️ Natural voice interaction
* 🧭 Real-time navigation assistance
* 👤 Walk With Me companion mode
* 📍 Smart path planning
* 🚨 Hazard detection and alerts
* 🔊 Voice-first user experience
* ⚡ Fast on-device processing
* ♿ Accessibility-focused interface

---

## 🎯 Problem Statement

Millions of visually impaired individuals face challenges while navigating unfamiliar environments. Existing solutions often require expensive dedicated hardware or provide limited environmental understanding.

SENTIA addresses this challenge by using AI to analyze surroundings in real time and deliver meaningful voice guidance using only a smartphone.

---

## 💡 Our Solution

Using computer vision and artificial intelligence, SENTIA continuously understands the user's surroundings, detects obstacles, identifies important objects, and provides clear spoken guidance. The application aims to improve confidence, reduce dependency, and enable safer independent mobility.
